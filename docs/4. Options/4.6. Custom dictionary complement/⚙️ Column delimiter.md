## Definitions

- `Tab` (default)
- `Comma`
- `Pipe`

## Examples

### Tab

```tsv
😄		@smile	@laugh	@good
😐		@normal
```

### Comma

```csv
😄,,smile,@laugh,@good
😐,,@normal
```

### Pipe

```
😄||smile|@laugh|@good
😐||@normal
```

## Notes

- For now, there are no ways to escape `,` and `|`, so I recommend to use `Tab` if you register `,` or  `|`  with [[📚 Custom dictionaries]]
